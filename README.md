# pdf-reader 

###### pdf reader is a web app that allows you to read pdf files

## Features

- [ zooming ] 
- [ navigating ] 
- [multiple tabs for opening / reading multiple pdf's]
- [ searching ] 
- [ bookmarking ] 
- [ highlighting ] 
- [ commenting ] 
- [ exporting ] 
- [ printing ] 
- [ password protected ] 
- [ multi language ] 
- [ dark mode ] 
- [ light mode ] 
- [ two way audio ] 
- [ two way video ] 
- [ two way voice ]

## Installation

```bash
git clone https://github.com/ArifMiah07/pdf-reader.git
cd pdf-reader
npm install
```

## Usage

```bash
npm start
```
or

```bash
npm run dev
```
